Hash4

A tool for publishing a blog. No database required.

    A simple way to publish your blog, without managing a database.

    To get started, create a posts.txt file in the same folder as your index.php.

    The first two lines of posts.txt tell Hash4 some infomration about the blog itself.

    Site Name My Tumblelog
Author Ankur Taxali

    The rest of the lines are for your posts. Posts are seperated by 4 hashes. 

    ####
I invented Hash4!
3/18/13
Hash4 is a blogging system anyone can use. 
Right now it is configured for very simple writing. 
Line breaks are supported!
####

    Hash4 also supports Wordpress-style Template Tags for templating.

    Retreive posts

    get_posts();

    Returns

    Array of Posts

    Example usage

    foreach($posts as $id => $post){ 
	$postdata = set_postdata($post);
	?>
	<li class="post"> 
		<h3>
		<a href="<?php the_permalink($id); ?>">
		<?php echo $postdata['title'] ?> 
		</a>
		</h3>
		<em><?php echo $postdata['date']; ?></a></em>
		<?php if (is_single() ) {
			echo "<div class='post-content'>"; 
			the_content($postdata); 
			echo "</div>"; 
		}
		?>
		</p>
	</li>
	<?php
} ?>

    Retreive info about the blog

    get_bloginfo();

    Perameters

    • 'tag' => returns the tag line
    • 'author' => returns the author
    • 'title' => returns the site title

    Example usage

    echo get_bloginfo('tag');
